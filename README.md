# Flask-sample
## Setup
1. Install Flask
```
pip install Flask
```
2. Download code
```
git clone https://github.com/n-yoshikawa/flask-sample
```
3. Run code
```
cd flask-sample
flask run --debug
```
Open `http://127.0.0.1:5000/` in your browser.

4. Read the code, learn, develop!

Tested in Python 3.8.10, Flask 2.2.3
